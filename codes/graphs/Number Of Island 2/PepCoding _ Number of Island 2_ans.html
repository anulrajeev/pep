Number of Island 2

Hey coder! All set to get started with the problem, "Number of Island 2"? Great, let's begin.

Reader, make sure that you thoroughly go through the problem to understand the upcoming discussion well.


Important Links : Problem Link, Solution Video


Problem Discussion
  • You are given a m*n matrix mat.
  • Originally, the 2D matrix has only 0s which means there is only sea in the matrix.
  • Each of the k lines in the given list contains two numbers denoting the position of the cell (row, col) in the matrix which is to be changed to 1 (or land).
  • You are required to return how many islands are there in the matrix after each operation. This output is returned in the form of an array of size K.

Lets understand it through the following example.

In figure 1, we are given a 2d matrix and a list of positions which are to be converted to land.

  • When (0,3) is made a land then the number of islands is [1].
  • Then, when (1,2) is flipped, the number of islands becomes 2, hence the array is [1,2].
  • Now coming to (0,2), all the lands form a single island, so the array is [1,2,1].
  • Next for (1,4), one more island is added hence [1,2,1,2].
  • For (2,3), the total islands are 3 i.e. [1,2,1,2,3].
  • At last, for (1,3), we get a single island, hence, [1,2,1,2,3,1].

The output for the given input will be [1,2,1,2,3,1].

Approach

To solve this problem we don't use the DFS (Depth First Search) method. This is because after converting each element in the given list, we will have to do work which takes n2 time. Hence, if there are K elements in the list, then the time complexity of this method would be O(Kn2).

We can use a better approach which has a time complexity of O(n2+K) to solve this problem.

We will make a parent array for the DSU (Disjoint Set Union) of the matrix.

The DSU of a 2D array is made on the basis of the cell number. The cell number of the cell (r,c) is calculated by the formula, r*col.length+c. For example, in a 4*5 matrix, the cell number of the cell (1,3) is 1*5+3=8.

In this problem, a set is defined in terms of an island i.e. all the cells which are a part of the same island constitute the same set.

  • The DSU of the given matrix ranges from the cell number 0 to 14.
  • For every position to be changed into land as given in the list, we calculate its cell number.
  • At that cell number, we change the cell in the matrix to 1 (or land) and increase the counter variable "ans" by 1.
  • Now we need to check if the flipped cell combines with its neighbors (left, right, up, down) to form an island or not.
  • If this flipped land cell doesn't combine with any other islands, then an increase in "ans" was valid.
  • However, if the flipped cell combines with any of its neighbor cells then they merge to form a single island on the basis of their rank and hence the "ans" counter is decremented by 1.
    The number of times the flipped cell merges with its neighbors to form a single island, that number of times 1 is decremented from "ans".
