Kruskal Algorithm

Important Links : Problem Link, Solution Video


Problem Discussion

In this problem you are given a number n, representing the number of vertices and there are edges in between some of the vertices, which is also given.

All you have to do is find the sum of edge weights of the minimum spanning tree.



For example:

Sample Input:

7
8
0 1 10
1 2 10
2 3 10
0 3 10
3 4 2
4 5 3
5 6 3
4 6 8

Sample Output:

38



How?

For the given input, the graph will look like:

And the corresponding MST will look like:

And the sum of weights of included edges is 38, which is the output.

Approach:
  • In this problem, we need to find the sum of edge weight MST (minimum spanning tree) using DSU (disjoint set union).
  • To get this, we need to convert the given graph into a minimum spanning tree, i.e. if there are 'v' vertices in a graph then we need to keep connected these 'v' vertices with 'v-1' edges such that the sum of edge weight is minimum.
  • First of all, we need to sort the given edges on the basis of the edge's weight. For example, consider the sample input.
  • And then we will travel the edges one by one in increasing order, and using DSU we will find the MST.
  • At last, add the weight of the edges used in the MST.

For more clarity of this part, watch this part of the video.

Let's try to code this!
  • To start with, define an array, 'edges' of type 'pair' and length edge.length.
  • Then using a 'for' loop, fill the 'edges' array by creating a pair for each edge.
  • After filling the edges array completely, sort the 'edges' array.
  • Now define a variable 'ans', initialized to '0' and two arrays, 'parent' and 'rank' of length 'n+1' (where n is the number of vertices).
  • Then using a 'for' loop initialized with i = 0, put the value 'i' at parent[i] and '1' at rank[i], until 'i' remains less than 'parent' array's length.
  • After that, again run a 'for' loop, with 'i' initialized to 0 until 'i' remains less than edges.length.
  • On entering the 'for' loop, capture the starting vertex (u), ending vertex (v) and weight of edge between these vertex (wt) present at ith index in 'edges' array.
  • Now initialize a 'flag' of type Boolean and store the result of union(u, v) in it.
  • And if the flag is false then it means that either 'u' or 'v' is not part of the union yet. So we will consider this edge and therefore add it's weight to 'ans'.
  • And at last return 'ans'.
  • To code union function, pass two integers, representing the two vertices of an edge.
  • Then store the leader of 'x' and 'y' in variables 'lx' and 'ly'. Check whether both of them are equal or not. If yes, then return true.
  • Otherwise, check if the rank of 'lx' is greater than the rank of 'ly'. If yes then put 'lx' in the parent of 'ly'. If not, then check if the rank of 'lx' is smaller than the rank of 'ly', if yes then put 'ly' in the parent of 'lx'. Otherwise put 'lx' in parent[lx] and increment rank[ly] by 1.
  • And at last return false.
  • To find the leader of any vertex, check if the parent of that vertex is the vertex itself and if it is then return that vertex.
  • Otherwise, call the find(parent[x]) and store the result in a variable, say 'temp'. Put the 'temp' in parent[x]. And finally return 'temp'.

import java.util.*; import java.io.*; public class Main { public static void main(String[] args) throws Exception { BufferedReader br = new BufferedReader(new InputStreamReader(System.in)); int v = Integer.parseInt(br.readLine()); int e = Integer.parseInt(br.readLine()); int[][] edges = new int[e][3]; for (int i = 0; i < e; i++) { String[] st = br.readLine().split(" "); edges[i][0] = Integer.parseInt(st[0]); edges[i][1] = Integer.parseInt(st[1]); edges[i][2] = Integer.parseInt(st[2]); } System.out.println(minCostToSupplyWater(v, edges)); } static int[] parent; static int[] rank; public static class Pair implements Comparable< Pair> { int u; int v; int wt; Pair(int u, int v, int wt) { this.u = u; this.v = v; this.wt = wt; } @Override public int compareTo(Pair o) { return this.wt - o.wt; } } public static int minCostToSupplyWater(int n, int[][] pipes) { Pair[] edges = new Pair[pipes.length]; for (int i = 0; i < pipes.length; i++) { int u = pipes[i][0]; int v = pipes[i][1]; int wt = pipes[i][2]; edges[i] = new Pair(u, v, wt); } int ans = 0; Arrays.sort(edges); parent = new int[n + 1]; rank = new int[n + 1]; for (int i = 0; i < parent.length; i++) { parent[i] = i; rank[i] = 1; } for (int i = 0; i < edges.length; i++) { int u = edges[i].u; int v = edges[i].v; int wt = edges[i].wt; boolean flag = union(u, v); if (flag == false) { ans += wt; } } return ans; } public static int find(int x) { if (parent[x] == x) { return x; } int temp = find(parent[x]); parent[x] = temp; return temp; } public static boolean union(int x, int y) { int lx = find(x); int ly = find(y); if (lx == ly) { return true; } if (rank[lx] > rank[ly]) { parent[ly] = lx; } else if (rank[lx] < rank[ly]) { parent[lx] = ly; } else { parent[lx] = ly; rank[ly]++; } return false; } }
